# 🛒 Ecommerce Sales Analysis Dashboard

An interactive **Ecommerce Sales Analysis Dashboard** developed using **Microsoft Excel** to analyze sales performance, customer behavior, product trends, and business profitability. This dashboard helps businesses make data-driven decisions through clear visualizations and KPIs.

---

## 📌 Project Overview

This project analyzes ecommerce sales data to identify key business insights, monitor sales performance, and understand customer purchasing behavior. The dashboard is designed to provide an easy-to-understand overview of important business metrics.

## 🚀 How to Use

1. Clone this repository.

```bash
git clone https://github.com/mountusaini/Sales In Excel Data Analyst.git
```

2. Open **Ecommerce Sales Analysis.xlsx** using Microsoft Excel.

3. Enable editing if prompted.

4. Use the slicers and filters to interact with the dashboard.
